# 🟥 **P0 — Critique / Must Have (MVP)**

### **📱 Application Mobile – Consommateurs**
- P0 : En tant que consommateur, je veux scanner un produit pour obtenir instantanément son score éthique et environnemental.  
- P0 : En tant que consommateur, je veux consulter un résumé clair du score du produit.  
- P0 : En tant que consommateur, je veux consulter les détails du score (traçabilité, impact carbone, certifications, conditions de travail).  
- P0 : En tant que consommateur, je veux voir un code couleur (vert, orange, rouge).  
- P0 : En tant que consommateur, je veux consulter la fiche complète d’un produit.  
- P0 : En tant que consommateur, je veux recevoir des alternatives plus éthiques si un produit a un mauvais score.  
- P0 : En tant que consommateur, je veux définir mes préférences (bio, local, vegan, équitable).  
- P0 : En tant que consommateur, je veux consulter mon historique de scans.  

### **🌐 Producteurs – Site Web**
- P0 : En tant que producteur, je veux ajouter un nouveau produit avec toutes ses informations.  
- P0 : En tant que producteur, je veux modifier les informations d’un produit existant.  
- P0 : En tant que producteur, je veux supprimer un produit.  
- P0 : En tant que producteur, je veux enregistrer chaque étape de production sur la blockchain.  
- P0 : En tant que producteur, je veux uploader mes certifications officielles.  

### **🛡️ Administrateurs**
- P0 : En tant qu’administrateur, je veux gérer les comptes des consommateurs et producteurs.  
- P0 : En tant qu’administrateur, je veux valider les comptes producteurs.  
- P0 : En tant qu’administrateur, je veux vérifier les certifications uploadées.  

### **⚙️ Techniques**
- P0 : En tant que système, je dois enregistrer les données de traçabilité sur la blockchain.  
- P0 : En tant que système, je dois générer un identifiant unique pour chaque produit.  
- P0 : En tant que système, je dois assurer une authentification sécurisée.  
- P0 : En tant que système, je dois chiffrer les données sensibles.
